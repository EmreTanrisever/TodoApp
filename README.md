# TodoApp
Thanks to this application, you can create To-Do's and organize your To-Dos. if you have finished the task you added, you can delete it by pressing the delete button.

## KeyWord
<ul>
    <li>MVVM</li>
    <li>TableView</li>
    <li>Codable</li>
    <li>SPM</li>
    <li>SnapKit</li>
    <li>UserDefaults</li>
</ul>
